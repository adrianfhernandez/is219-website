# Database Security Professional Portfolio

This is a professional portfolio website showcasing expertise in database security, inspired by Jeffrey A. Hoffer's work in database systems. The portfolio demonstrates the professional direction towards database security, highlights key skills and projects, and illustrates the integration of AI in security workflows.

## Features

- **Professional Direction**: Focused on database security specialization
- **Skills Showcase**: Technical and soft skills in database security
- **Project Portfolio**: Examples of security-related projects including Rate My Professor search interface
- **AI Integration**: How AI enhances database security workflows
- **Responsive Design**: Professional, clean interface

## Setup and Installation

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)
- Node.js 16 or higher (for RMP components)
- npm (Node.js package manager)

### Installation

1. Clone or download this repository
2. Navigate to the project directory
3. Install dependencies:

```bash
pip install -r requirements.txt
```

### Running the Application

To start the portfolio website:

```bash
python main.py
```

The website will be available at `http://localhost:5000`

### Running the Rate My Professor Components

The portfolio includes a Rate My Professor search interface as a featured project. To run the RMP components:

1. Install Node.js dependencies for RMP:
```bash
cd rmp
npm install
```

2. Start the RMP API server (in a separate terminal):
```bash
cd rmp
npm run api-server
```

The RMP API server will run on `http://localhost:3001`

3. Open the RMP interface by clicking "View Projects" in the portfolio or navigate to `rmp/scripts/rmp_ui.html`

### Docker Deployment

If you prefer to run with Docker:

```bash
docker-compose up
```

## Generate SSH Keys and Connect to GitHub

> Only do this once per machine.

1. Generate a new SSH key:

```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
```

(Press Enter at all prompts.)

2. Start the SSH agent:

```bash
eval "$(ssh-agent -s)"
```

3. Add the SSH private key to the agent:

```bash
ssh-add ~/.ssh/id_ed25519
```

4. Copy your SSH public key:

- **Mac/Linux:**

```bash
cat ~/.ssh/id_ed25519.pub | pbcopy
```

- **Windows (Git Bash):**

```bash
cat ~/.ssh/id_ed25519.pub | clip
```

5. Add the key to your GitHub account:
   - Go to [GitHub SSH Settings](https://github.com/settings/keys)
   - Click **New SSH Key**, paste the key, save.

6. Test the connection:

```bash
ssh -T git@github.com
```

You should see a success message.
